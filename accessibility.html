Accessibility Statement

Last updated: July 6, 2026

Commitment

shaya.click aims to provide a clear, accessible professional profile that works across desktop, tablet and mobile devices and remains usable with keyboard navigation, semantic HTML and readable contrast.

Current Measures

  • Semantic page landmarks, headings and navigation labels.
  • Responsive layout for mobile and desktop screens.
  • Visible focus states, skip link and reduced-motion support.
  • Alternative text for profile imagery.

Known Limitations

External services linked from this website, including LinkedIn, Google Scholar, ResearchGate, GitHub and YouTube, are controlled by their respective providers and may have different accessibility behavior.

Feedback

If you find an accessibility problem on this website, contact Shaya Pourmirza through the public LinkedIn profile linked from the homepage. Please include the page URL, device/browser context and a short description of the issue so it can be reviewed efficiently.
